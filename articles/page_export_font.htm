Export Font Page

Glyphr Studio is able to export a project to two different Font File formats: Open Type (.otf) fonts and SVG Fonts (.svg). More information about specific file formats can be found in the File Formats article.

If you want to, there are options on the Project Settings Page that will allow you to pin an 'Export OTF' and/or an 'Export SVG' buttons on the Nav Bar. If you find yourself often exporting a font, these buttons are more accessible than navigating to the Export Font page.
