Attributes Panel

(Character Edit, Ligatures, Linked Shapes, Kerning, and Test Drive Pages)

The Attributes Panel is comprised of many sub-panels, depending on what is selected. If no shapes are selected, it displays the Character Panel, with character level options, and bulk-edit options for all shapes in that character.

If a shape or path point is selected, the Attributes Panel will display the Shape Panel, the Path Point Panel, and/or the Linked Shape Instance Panel.

The Attributes Panel also shows the Actions Panel, which displays actions that can be taken on the currently selected shape or point. There are also a few global actions that can be taken even if nothing is selected.
