Welcome!

This is help content for Glyphr Studio, Version 1. If you are brand new, read through the Navigation and Layout article. Then, dive into glyphrstudio.com/v1/online to get going.

If you looked through the help content and didn't find the answer you're looking for, feel free to get in touch with us! Or, you could help improve our content by filing an issue over at GitHub.

Tutorial!

If you are a first time typeface designer, be sure to read through our tutorial over at glyphrstudio.com/v1/tutorial. The tutorial focuses on setting up another vector design program so that glyph outlines can be easily imported via the Glyphr Studio Import SVG page.
